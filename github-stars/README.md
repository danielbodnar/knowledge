# GitHub Stars Index

This directory contains starred GitHub repositories organized by category.

## Categories

Repositories are organized into the following categories:

- **Development Tools** - IDEs, editors, development utilities
- **Frameworks** - Web frameworks, mobile frameworks, etc.
- **Libraries** - Utility libraries, UI components
- **DevOps** - CI/CD, infrastructure, deployment tools
- **Security** - Security tools, scanners, vulnerability databases
- **Data** - Databases, data processing, analytics
- **AI/ML** - Machine learning libraries, AI tools
- **Cloud** - Cloud-specific tools and utilities
- **Learning** - Educational resources, tutorials, examples
- **Other** - Miscellaneous repositories

## Adding Stars

Stars are automatically imported and categorized from your GitHub account.

You can also manually add repository information using the template from `/templates/github-star-template.md`.

## Automatic Updates

The repository is configured to periodically sync with your GitHub stars and update classifications automatically.
