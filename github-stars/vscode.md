# Visual Studio Code

## Description

Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js.

## Why Starred

The most popular code editor for modern development. Excellent extension ecosystem, integrated terminal, Git support, and debugging capabilities. Essential tool for daily development work.

## Key Features

- IntelliSense code completion
- Built-in Git integration
- Integrated terminal
- Extensive extension marketplace
- Multi-language support
- Remote development (SSH, Containers, WSL)
- Live Share for collaboration
- Customizable and themeable
