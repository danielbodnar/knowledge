# Bookmarks Index

This directory contains browser bookmarks organized by source and category.

## Structure

- `chrome/` - Chrome browser bookmarks
- `firefox/` - Firefox browser bookmarks
- `general/` - General/uncategorized bookmarks

## Categories

### Development
- Programming languages and frameworks
- Development tools
- Code editors and IDEs

### DevOps
- CI/CD tools
- Infrastructure as Code
- Monitoring and logging

### Security
- Security tools
- Best practices
- Vulnerability databases

### Design
- UI/UX resources
- Design systems
- Color palettes and fonts

### Data
- Databases
- Data visualization
- Analytics platforms

### AI/ML
- Machine learning frameworks
- AI tools and services
- Research papers

### Cloud
- AWS, Azure, GCP resources
- Cloud architecture patterns
- Serverless computing

### Productivity
- Task management
- Note-taking apps
- Automation tools

### Learning
- Online courses
- Tutorials
- Documentation

### Reference
- API references
- Specifications
- Standards documentation

## Adding Bookmarks

Use the bookmark template from `/templates/bookmark-template.md` to add new bookmarks.

Bookmarks are automatically categorized and organized by the AI classification system on each commit.
