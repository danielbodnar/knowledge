# Cheatsheets Index

This directory contains technical cheatsheets and quick references for various technologies, tools, and concepts.

## Available Cheatsheets

Cheatsheets are organized by topic and include:

- Programming languages (Python, JavaScript, Go, Rust, etc.)
- Command-line tools (Git, Docker, Kubernetes, etc.)
- Frameworks and libraries
- Cloud platforms (AWS, Azure, GCP)
- Databases (SQL, NoSQL)
- Development tools
- Best practices and patterns

## Format

All cheatsheets follow a consistent format:
- Quick reference section at the top
- Categorized commands/syntax
- Examples and use cases
- Tips and tricks
- Related resources

## Adding Cheatsheets

Use the template from `/templates/cheatsheet-template.md` to create new cheatsheets.

## Contributing

Cheatsheets are continuously updated and expanded. The AI classification system helps maintain consistency and organization.
